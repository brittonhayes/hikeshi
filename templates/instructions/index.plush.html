Instructions

Submitting an incident

To submit your first incident, click the Create New dropdown button in the navigation bar. This will open up a form where you can break down exactly what happened.

Creating an user

To submit your first user, click the Create New dropdown button in the navigation bar and then click the User button.

Toggling Darkmode

To enable darkmode, click the  icon in the top right of the navigation bar.

To disable darkmode, click the   icon in the same position.

About Hikeshi

Hikeshi is a security incident response application that keeps documenting incidents simple, so you can focus on fighting fires. The name comes from the firefighting system established in Japan during the Edo period

Incident response involves a lot of "firefighting" in the sense that you're only called upon when something has gone very wrong. In high stress situations, it's nice to have a simple tool that assists you while you fight those fires.

The UI is not cluttered. All the colors are easy on the eyes with darkmode support by default. It isn't feature packed, but it does exactly what it needs to while you handle the situations at hand.
